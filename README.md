# RPG Python

https://replit.com/@yyuiiuy55/Tavern-Master

Projeto desenvolvido utilizando Programação Orientada a Objetos.

## Objetivo

Simular um RPG simples aplicando:

- Encapsulamento
- Herança
- Polimorfismo
- Abstração

## Tecnologias

| Biblioteca | Versão |
|------------|---------|
| Python | 3.12 |
| abc | Biblioteca padrão |
| json | Biblioteca padrão |
| os | Biblioteca padrão |

## Estrutura

```text
models/
services/
repositories/
interface/
data/
```

## Funcionalidades

- Criação de herói
- Sistema de combate
- Evolução de personagem
- Sistema de missões
- Persistência em JSON

## Conceitos de POO

### Encapsulamento

Atributos privados com getters.

### Herança

Heroi e Monstro herdam de Personagem.

### Polimorfismo

Método atacar() implementado nas subclasses.

### Abstração

Classe abstrata Personagem.

## Como executar

```bash
python main.py
```
